มาทำเว็บไทยให้ กันเถอะ

เราสนับสนุนให้เว็บไทยเปลี่ยนเป็น HTTPS

ความตั้งใจของ Secure The Web Thailand

ในแวดวงการพัฒนาเว็บไซต์ในเมืองนอก กำลังมีการผลักดันให้เว็บไซต์ ต่างๆติดตั้ง SSL เพื่อเป็นการปกป้องข้อมูลผู้ใช้เพื่อเป้นการสร้างความมั่นใจได้ว่าข้อมูลต่างๆขอผู้ใช้เว็บไซต์จะไม่ถูกขโมยได้ โดยในปัจจุบันกลุ่มผู้พัฒนา Browser เริ่มทำการปิดกั้นไม่ใช้เว็บไซต์ที่ไม่ได้ติดตั้ง SSL เข้าถึงคุณสมบัติใหม่ๆได้ และเพื่อเป็นการสนับสนุให้เว็บไซต์ในประเทศไทยเปลี่ยนผ่านไปสู่ ยุค HTTPS เรา Secure The Web Thailand ขอเป็นตัวแทนในการผลักดันให้เรื่องนี้เกิดขึ้นในไทยอย่างจริงจัง

ทำไมต้อง HTTPS?

มั่นใจว่าข้อมูลของผู้ใช้จะปลอดภัย

การติดตั้ง SSL จะทำให้ผูเจ้าของเว็บไซต์มั่นใจได้ว่าข้อมูลของผู้ใช้จะไม่ถูกดักจับจากผู้อื่นได้ระหว่างทาง

Browser จะไม่เตือนว่าเว็บไซต์ของคุณไม่ปลอดภัย

ใน Browser รุ่นใหม่ๆ ถ้าผู้ใช้เข้าเว็บไซต์ที่ไม่ได้ติดตั้ง SSL จะมีคำเตือนมาบอก User ทันที่ว่าเว็บไซต์นี้ไม่ปลอดภัย

มีผลต่อ SEO

Search Engine หลายๆเจ้ากำลังจะเริ่มให้ความสำคัญกับ HTTPS และจะนำมานับรวมในคะแนนผลการค้นหาเร็วๆนี้

ได้ใช้ Feature ใหม่ๆ

Feature ใหม่ๆใน มาตรฐาน HTML จะไม่สามารถใช้งานบนเว็บไซต์ ที่ไม่ได้ติดตั้ง SSL ได้ รวมถึง Feature เก่าๆจะเริ่มไม่สามารถใช้งานบน HTTP ได้เช่นเดียวกัน
